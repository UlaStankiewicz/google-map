# google-maps
Google maps with React and Express

# dev deployment

## Prerequisites
node, npm, docker, docker-compose

## postgres 
docker setup  
``sh compose.sh dev``  
or set up pg using config in server/knexfile.ts

## server
``cd server``  
``npm i``  
``npm start-dev``  

## client
``cd client``  
``npm i``  
``npm start``
